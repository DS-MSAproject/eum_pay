1) Document Upload

POST /api/v1/rag/documents

Content-Type: multipart/form-data

  • file (required): .pdf | .docx | .txt
  • document_id (optional)
  • category (optional)
{
  "documentId": "petfood-openbook-v1",
  "filename": "petfood-openbook-upload-ready.pdf",
  "category": "FAQ",
  "status": "pending",
  "chunkCount": 0,
  "message": "Document accepted. Parsing and indexing started asynchronously."
}

2) Document Status (Polling)

GET /api/v1/rag/documents/{documentId}

{
  "documentId": "petfood-openbook-v1",
  "filename": "petfood-openbook-upload-ready.pdf",
  "category": "FAQ",
  "status": "processed",
  "errorMessage": null,
  "chunkCount": 31,
  "createdAt": "2026-04-14T03:00:00Z",
  "updatedAt": "2026-04-14T03:00:20Z"
}

Status values: pending | parsing | embedding | indexing | processed | failed

  2. Poll status every 2-3 seconds
  3. Enable search/chat only after processed
